# Smart School Bell System

## Description
A web-based automated school bell system built using Flask.

## Features
- Automatic bell scheduling
- Web dashboard
- Add/Delete schedules
- Manual bell trigger
- History logging

## Tech Stack
Python, Flask, HTML, CSS, JavaScript

## How to Run
pip install flask
python app.py
